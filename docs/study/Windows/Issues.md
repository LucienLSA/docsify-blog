
## win 11 电脑睡眠后再登录，桌面图标、任务栏消失的解决方法

此问题可能是windows 资源管理器崩溃了。可以不重启电脑进行恢复：

1. 同时按住键盘上的Ctrl +Alt+Delete这三个键(或者ctrl +shift +Esc)，出现任务管理器；
2. 点击 文件- 运行新任务；
3. 在创建新任务的输入框中输入“explorer.exe”,然后点击确定
